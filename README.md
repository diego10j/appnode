# appnode
App Actividad 1 de la materia de Ingeniería y Desarrollo en la Web.

## Herramientas

* Node
* Express
* MongoDB

## Instalación

```sh
git clone https://github.com/diego10j/appnode
npm install
npm start
```
Abrir en el navegador [http://localhost:3000/](http://localhost:3000/)
